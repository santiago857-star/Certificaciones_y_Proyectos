*proyectos*

Este repositorio tambien contiene una colección de proyectos desarrollados durante mi proceso de aprendizaje en desarrollo web y programación en Python. 
Los proyectos presentados abarcan tanto aplicaciones web como programas de consola, y tienen como objetivo fortalecer habilidades en diseño, lógica de programación y resolución de problemas.

Uno de los proyectos principales es una página web informativa sobre agujeros negros, la cual presenta contenido educativo relacionado con la astronomía, explicando qué son los agujeros negros, sus tipos y algunas características relevantes. Este proyecto cuenta con un diseño visual atractivo, una estructura organizada y elementos interactivos que mejoran la experiencia del usuario. Fue desarrollado utilizando HTML, CSS y JavaScript.

Otro proyecto corresponde a una tienda virtual, la cual simula el funcionamiento básico de un comercio electrónico. En ella se muestran productos con sus respectivos precios y una interfaz clara e intuitiva para el usuario. El enfoque de este proyecto está en la estructura y el diseño visual de la página, utilizando HTML y CSS.

También se incluye un proyecto llamado Paseo por el Parque, que consiste en una página web visual cuyo objetivo principal es representar gráficamente un recorrido por un parque. Este proyecto se centra en la maquetación, el uso creativo de estilos y la presentación visual de los elementos, utilizando HTML y CSS.

Además de los proyectos web, el repositorio contiene varios proyectos desarrollados en Python. Uno de ellos es un sistema de gestión para una escuela, diseñado para manejar una base de datos que permite la asignación de clases, profesores y alumnos, así como la organización de horarios. Este proyecto se enfoca en la estructuración de datos y la lógica necesaria para administrar la información académica.

Otro proyecto en Python es la implementación del juego de Triqui (Tres en Raya), el cual se ejecuta en consola y permite a dos jugadores turnarse para jugar, validando las jugadas y determinando si existe un ganador o un empate. Este proyecto refuerza el uso de condicionales, ciclos y manejo de entradas del usuario.

Finalmente, se incluye un programa que simula el lanzamiento de dados, generando valores aleatorios y calculando la suma total de los resultados obtenidos. Este proyecto está orientado a la práctica de la generación de números aleatorios y operaciones básicas en Python.

En conjunto, estos proyectos reflejan mi proceso de aprendizaje y desarrollo de habilidades en programación, diseño web y lógica computacional, y forman parte de mi formación académica en Ingeniería en Sistemas.
